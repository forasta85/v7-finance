## ⚡ A FORMA MAIS RÁPIDA (5 minutos)

### **Passo 1: GitHub (3 min)**

**Windows:**
```cmd
setup-github.bat
```

**Mac/Linux:**
```bash
bash setup-github.sh
```

O script vai:
- ✅ Configurar Git
- ✅ Abrir GitHub para você criar repositório
- ✅ Pedir URL do repo
- ✅ Fazer push automático

### **Passo 2: Vercel (2 min)**

O script abre Vercel automaticamente.

No Vercel:
1. Importar repositório
2. Adicionar 2 variáveis de ambiente (o script mostra quais)
3. Clicar em Deploy

**PRONTO! ✅**
